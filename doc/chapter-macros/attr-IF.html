IF=...

Conditionally include element.

Must be in the form of {l}... IF='[expr]' ...{r}, where:

  • [expr] represents the expression that is evaluated to a boolean.


If the expression expr evaluates to true, the element is included in the final document; otherwise it is discarded. This attribute macro can also be found on almost all macro elements, where it performs the same function with a small caveat:

  • Evaluation of an element macro is terminated only after this attribute macro is encountered and evaluated to false; any previous attributes can already have an effect on the final document. If such behaviour is unwanted (for example in a {l}SET/{r} macro), the IF attribute should be written before all others.


See also: attribute ELIF, attribute ELSE, expressions

Example

input output
{l}SET n='1'/{r}
{l}p IF='n==1'{r}one{l}/p{r}
{l}p ELIF='n==2'{r}two{l}/p{r}
{l}p ELSE{r}none{l}/p{r} 		{l}p{r}one{l}/p{r}
