ELIF=...

Include element if previous condition atribute macro fails.

Must be in the form of {l}... ELIF='[expr]' ...{r}, where:

  • [expr] represents the expression that is evaluated to a boolean.


The element is included in the final document only if an attribute macro IF or ELIF on a previous element evaluates to false, and the expression expr evaluates to true. This attribute macro can also be found on almost all macro elements, where it performs the same function with a small caveat:

  • Evaluation of an element macro is terminated only after this attribute macro is encountered and is determined, that the element should be discarded. Any previous attributes can already have an effect on the final document. If such behaviour is unwanted (for example in a {l}SET/{r} macro), the ELIF attribute should be written before all others.


See also: IF, ELSE, expressions

Example

input output
{l}SET n='2'/{r}
{l}p IF='n==1'{r}one{l}/p{r}
{l}p ELIF='n==2'{r}two{l}/p{r}
{l}p ELSE{r}none{l}/p{r} 		{l}p{r}two{l}/p{r}
