{l}ELSE{r}...{l}/ELSE{r}

Includes child elements if previous condition macros fail.

Must follow a previous {l}IF{r} or {l}ELIF{r} macro.
Child elements are included in the final document only if a previous {l}IF{r} or {l}ELIF{r} macro failed.


See also: {l}IF{r}, {l}ELIF{r}, {l}SET{r}, attribute ELSE, expressions

Example

input output
{l}SET lang="en"/{r}

{l}IF lang="de"{r}
{t}{l}p{r}Deutsch{l}/p{r}
{l}/IF{r}
{l}ELSE{r}
{t}{l}p{r}English{l}/p{r}
{l}/ELSE{r}
 {l}p{r}English{l}/p{r}
