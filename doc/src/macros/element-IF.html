{l}IF{r}...{l}/IF{r}

Conditionally includes child elements.

Must be in the form of {l}IF [cond]{r}, where [cond] can either be a boolean check TRUE=[expr] or FALSE=[expr], or a direct variable comparison [var]=[value]. Direct variable comparison is equivalent to TRUE='[var]==[value]'.
Child elements are included in the final document only if all conditionals [cond] are evaluated to true.


See also: {l}ELIF{r}, {l}ELSE{r}, {l}SET{r}, attribute IF, expressions

Example

input output
{l}IF TRUE='6>3'{r}
{t}{l}p{r}6 is greater than 3{l}/p{r}
{l}/IF{r}
 {l}p{r}6 is greater than 3{l}/p{r}
{l}SET lang="en"/{r}
{l}IF lang="en"{r}
{t}{l}p{r}English{l}/p{r}
{l}/IF{r}
 {l}p{r}English{l}/p{r}
