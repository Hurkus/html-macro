INCLUDE=[path]

Include contents of another file as child elements of the current element.

Works in the same way as the element macro {l}INCLUDE{r}, with the exception that included content is never wrapped (NO-WRAP) and the user cannot declare any local variables.


The file is included immediately after encountering the attribute. This means that some attributes might not have fully coppied to the final document. This would affect macros such as {l}DEL-ATTR{r}, as they would not be able to remove attributes that do not exist yet. If such behaviour is unwanted, this macro should be written after all others.


See also: {l}INCLUDE{r}

Example

content.html
{l}IF lang="de"{r}
{t}{l}p IF='i==1'{r}Ein{l}/p{r}
{t}{l}p ELIF='i==2'{r}Zwei{l}/p{r}
{t}{l}p ELIF='i==3'{r}Drei{l}/p{r}
{l}/IF{r}
{l}ELSE{r}
{t}{l}p IF='i==1'{r}One{l}/p{r}
{t}{l}p ELIF='i==2'{r}Two{l}/p{r}
{t}{l}p ELIF='i==3'{r}Three{l}/p{r}
{l}/ELSE{r}
input output
{l}SET lang="en"/{r}
{l}body lang="\{lang}"{r}
{t}{l}ol{r}
{t*2}{l}FOR i='1' TRUE='i{l}=3' i='i+1'{r}
{t*3}{l}li INCLUDE="content.html"/{r}
{t*2}{l}/FOR{r}
{t}{l}/ol{r}
{l}/body{r} 		{l}body lang="en"{r}
{t}{l}ol{r}
{t*2}{l}li{r}
 {t*3}{l}p{r}One{l}/p{r}
 {t*2}{l}/li{r}
{t*2}{l}li{r}
 {t*3}{l}p{r}Two{l}/p{r}
 {t*2}{l}/li{r}
{t*2}{l}li{r}
 {t*3}{l}p{r}Three{l}/p{r}
 {t*2}{l}/li{r}
{t}{l}/ol{r}
{l}/body{r}
